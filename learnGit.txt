这是一个用来学习Git命令文本文档
已学习命令：
git add . *将当前修改添加到暂存区*
git commit -m "填写此次修改的备注" *讲暂存区的内容提交到仓库中*
git status *查看当前git的状态*
git log *查看git的日志*
git checkout [用来绝对定位版本的唯一编码] *回到对应编码的版本，后跟减号代表回退上一个版本*
*注意：checkout命令本质上是一个切换命令，版本回退是具体应用*
git tag -a(注释) [标签名] -m "标签的备注" *为当前提交内容打一个标签*
git tag *查看所有标签*
git tag -a [标签名] -m "标签的备注" [用来绝对定位版本的唯一编码] *为对应编码的提交打一个标签*
git show [标签名] *查看此标签的详细信息*
git branch [分支名] *创建新的分支*
git checkout [分支名] *切换到对应的分支上*
git merge [分支名] *将对应的分支合并*
*注意：合并分支是将所选分支合并到当前分支上*
git remote add [远程仓库名] [远程仓库地址] *将本地仓库连接到远程仓库上*
git remote *查看所有已连接的远程仓库*
git push -u [远程仓库名] [分支名] *将代码提交到对应的远程仓库*
*注意：第一次向空的远程仓库push时要在分支名前面加上‘+’进行强制更新*
git pull *获取远程仓库的内容的更新*
*注意：出现冲突的时候push请求会被驳回，可以拉取最新的版本整合冲突之后再次push*
